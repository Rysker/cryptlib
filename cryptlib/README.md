# Cryptlib

Cryptlib is a simple library for ECC and CRYSTALS-Kyber KEM operations.  

It supports key generation, signing/verification, and key encapsulation mechanisms (KEM) for post-quantum cryptography.

**Supported algorithms:**
- ECC: P-256, P-384, P-521
- CRYSTALS-Kyber: 512, 768, 1024
